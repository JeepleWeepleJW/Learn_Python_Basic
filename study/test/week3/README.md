## 직각삼각형 변의 길이 구하기

직각삼각형 세변의 길이가 각각 x, y, z일 때 z 제곱의 길이를 구하시오
- x, y는 사용자에게 입력받는다.
- 함수를 활용해서 코드를 작성하세요,

```
입력 예시
x : 3
y : 4

출력 에시
z제곱 : 25

```

## 사각형, 삼각형 별찍기

```

출력 예시 : 
*
**
***
****
*****

입력 : 사각형

*****
```

# 시험 점수 평균 구하기

10명의 학생 점수를 입력받고 해당 점수의 평균을 출력하는 코드를 작성하세요.
- 함수를 활용해서 코드를 작성하세요.

```
입력 예시 :
1번 학생 점수 : 20
2번 학생 점수 : 60
...
10번 학생 점수 : 100

출력 예시 :
10명 학생 점수 평균 : 56
```
